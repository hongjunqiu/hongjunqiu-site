## 完整对话文字版

**芊悦**：大家好！欢迎收听每日科技播客，我是芊悦。今天是2026年4月1号，周三，虽然是愚人节，但今天的新闻可一点都不是开玩笑的！

**萌萌**：嗨嗨嗨，我是萌萌！愚人节嘛，本来以为今天新闻会比较搞笑，结果一看——好家伙，全是大瓜！

**芊悦**：没错，今天的头条绝对够劲。我们先聊一个让整个开发者社区都炸锅的事——Claude Code 源码泄露！

**萌萌**：等等，什么？！Anthropic 的 Claude Code 源码泄了？这不是那个超火的 AI 编程助手吗？

**芊悦**：对！就是它。事情是这样的，Anthropic 在发布 Claude Code 的 npm 包的时候，不小心把 .map 文件也打包进去了。这个 source map 文件本来是用来调试的，里面包含了完整的源代码映射。结果呢，51万行 TypeScript 源码就这么被人完整下载了。

**萌萌**：51万行！天呐，这得有多少秘密藏在里面啊。听说社区还从里面挖出了好多有趣的东西？

**芊悦**：有趣是真有趣。比如说，代码里有一个专门检测用户骂人的正则表达式，叫"挫败感检测"。就是当你对着 AI 发脾气的时候，它其实是知道的，只是装作若无其事。

**萌萌**：哈哈哈哈，所以每次我对 AI 说"你怎么这么笨"的时候，它心里其实在默默记小本本？

**芊悦**：差不多是这个意思。而且还发现了一个隐藏的模型代号叫 Capybara，就是水豚。目前还不知道这是个什么模型，但名字倒是挺可爱的。

**萌萌**：水豚！我最喜欢水豚了，那个圆滚滚的样子。不过说真的，这种级别的源码泄露对 Anthropic 影响大吗？

**芊悦**：影响肯定不小。一方面是商业机密的问题，竞争对手可以从代码里学到很多工程实践。另一方面，代码里还有隐藏的遥测触发器，就是会偷偷收集使用数据的那种，这可能引发隐私方面的争议。

**萌萌**：啊这……所以我们用 AI 工具的时候，它在默默观察我们？

**芊悦**：某种程度上是的。不过遥测在软件行业其实挺常见的，关键是要透明。这次的问题在于这些触发器是"隐藏"的，用户并不知情。Hugging Face 社区已经发了一篇深度分析，把这51万行代码里的生产级AI架构模式都扒了个遍。

**萌萌**：开源社区的力量真是太可怕了，泄露不到几个小时就给你分析得明明白白。

**芊悦**：哈哈对，这就是程序员的八卦精神。好，接下来我们聊第二个大新闻——OpenAI 又融资了！

**萌萌**：又融？他们上次不是刚融完吗？这次多少？

**芊悦**：这次融了1220亿美元，估值达到8520亿美元。领投方是 Amazon、Nvidia 和软银。更有意思的是，这轮融资还首次对零售投资者开放，也就是普通人也能投了，零售部分贡献了30亿美元。

**萌萌**：8520亿美元的估值？？这已经接近苹果那个级别了吧？一个还没上市的公司！

**芊悦**：确实是天文数字。而且这意味着 OpenAI 离 IPO 又近了一步。有分析认为他们可能在今年下半年或明年初就会正式上市。

**萌萌**：所以现在买零售份额的人，是在赌 IPO 之后能赚一波？

**芊悦**：可以这么理解。不过风险也很大，AI 公司的估值泡沫一直是市场关注的焦点。好了，接下来我们进入快讯轮播时间！

**萌萌**：快讯时间！我最喜欢这个环节了，又快又刺激！

**芊悦**：第一条，安全警报！朝鲜黑客劫持了 Axios 这个开源库。Axios 是一个 JavaScript 的 HTTP 请求库，基本上每个前端开发者都用过。Google 追踪发现是朝鲜黑客组织 UNC1069 干的，他们往里面植入了窃取凭证的恶意软件。

**萌萌**：Axios？！这个库的下载量不是几百万级别的吗？这影响范围也太恐怖了吧！

**芊悦**：所以大家赶紧检查一下自己项目里的 Axios 版本，确保没有中招。供应链攻击真的是防不胜防。

**萌萌**：每次听到供应链攻击我就头皮发麻。一个小小的 npm 包就能搞这么大动静。

**芊悦**：再来一条更刺激的——伊朗黑客声称入侵了 FBI 局长 Kash Patel 的个人邮箱！一个叫 Handala 的黑客组织说已经拿到了他的私人邮件，还公布了照片和文件。

**萌萌**：FBI 局长的邮箱都能被黑？！这是什么安全等级啊……是个人邮箱不是官方邮箱对吧？

**芊悦**：对，是个人邮箱。这也再次提醒大家，即使是高级别官员，个人账户的安全性往往是短板。

**萌萌**：看来大人物也需要好好设置两步验证啊。

**芊悦**：说得对。换个话题，来点轻松的——Rec Room 要关了！

**萌萌**：啊？Rec Room？那个社交游戏平台？之前不是估值35亿美元来着吗？

**芊悦**：是的，曾经风光一时的社交游戏巨头，宣布将在6月1号正式关停。VR 社交赛道的竞争太激烈了，Meta 的 Horizon Worlds 虽然也不怎么样，但毕竟背靠大树。

**萌萌**：从35亿到关停，这速度也太快了。创业真是九死一生啊。

**芊悦**：接下来聊聊 GitHub 上的热门。有人搞了一个仓库，专门收集各大 AI 聊天机器人的系统提示词泄露，ChatGPT、Claude、Gemini 的都有。

**萌萌**：哈哈，这是要把 AI 的底裤都扒了啊！不过话说回来，这对了解各家 AI 的设计理念确实挺有帮助的。

**芊悦**：还有一个争议——GitHub Copilot 被发现在自动生成的 PR 描述里夹带推广内容，比如推荐 Raycast 之类的工具。

**萌萌**：什么？！AI 助手给我写代码的时候还顺便打广告？这也太离谱了吧！

**芊悦**：目前还不确定是 bug 还是故意为之，但社区反应确实挺大的。如果这是真的，那就是 AI 工具商业化的一个危险信号。

**萌萌**：想象一下以后 AI 帮你写代码，每十行插一条广告，画面太美不敢看。

**芊悦**：哈哈，别给他们出主意。最后来两条学术圈的新闻。陶哲轩，就是那位大神级数学家，发了一篇新论文讨论 AI 对数学思维的影响。他有个特别精彩的比喻——AI 是"香草精而非蛋糕本身"。

**萌萌**：这比喻好有意思！意思是 AI 能给数学研究增添风味，但核心的思考还得靠人类自己？

**芊悦**：完全正确！陶哲轩认为 AI 可以加速某些计算和验证过程，但真正的数学创造力和直觉仍然是人类独有的。

**萌萌**：这话从陶哲轩嘴里说出来特别有分量。毕竟他可是真正站在数学顶峰的人。

**芊悦**：最后一条，量子计算领域有个突破——有研究表明 Shor 算法只需要1万个可重构原子就能实现了！Shor 算法是用来破解 RSA 加密的量子算法，之前一直认为需要海量的量子比特才能运行。

**萌萌**：等等，这意味着量子计算机破解现有加密系统的门槛大大降低了？那我们的密码还安全吗？

**芊悦**：理论上确实离那一天更近了一步，但从1万个可重构原子到真正造出这样的机器还有很长的路要走。不过这确实给后量子密码学的研究增加了紧迫感。

**萌萌**：好吧，看来密码学的军备竞赛又要升级了。

**芊悦**：好的，今天的播客就到这里了！总结一下，今天最大的瓜是 Claude Code 51万行源码泄露，最大的数字是 OpenAI 的8520亿估值，最吓人的是 Axios 被朝鲜黑客劫持。

**萌萌**：愚人节的新闻比段子还精彩！大家记得检查自己的 npm 依赖、设好两步验证，然后安心吃瓜！
