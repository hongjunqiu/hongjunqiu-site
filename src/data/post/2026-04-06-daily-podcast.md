## 完整对话文字版

**芊悦**：嘿大家好！欢迎收听《科技早知道》，我是芊悦～今天是4月6号，周一，新的一周开始啦。

**萌萌**：我是萌萌！哇今天的料真的太多了——Anthropic 居然开始限制 Claude Code 用户用 OpenClaw 了，还有一条更离谱的，微软说 Copilot "仅供娱乐用途"，你敢信？

**芊悦**：哈哈哈等等，我们慢慢聊。先说个我觉得最值得深入讨论的——Hacker News 上那篇火爆的文章，讲 AI 编程时代开发者技能退化，760 个赞、快 500 条评论。

**萌萌**：啊我看到了！标题特别扎心——"威胁在于你舒适地滑向不理解自己在做什么"。说实话，我觉得这标题就是在说我本人。

**芊悦**：其实这个话题挺严肃的。现在 AI 编程工具越来越强，很多人写代码就是 tab tab tab，自动补全接受一切。但你有没有想过，当你不再需要理解每一行代码的时候，你的编程能力其实在悄悄退化？

**萌萌**：嗯...这就好比你天天用导航开车，突然有一天导航坏了，你发现自己连回家的路都不认识了。

**芊悦**：这个比喻太准了。而且更可怕的是——你以为你还会开车，但其实你只是会跟着箭头走。

**萌萌**：但是我不太同意把这完全当成坏事诶。你想想，以前大家都手动管理内存，现在有垃圾回收了，这不也是一种"技能退化"吗？但效率提高了多少？

**芊悦**：这两件事不太一样吧。垃圾回收是把底层细节抽象掉了，但你作为开发者还是理解程序逻辑的。AI 编程的问题是，它可能连逻辑层面都帮你做了，你就变成了一个代码审查员——而且可能还是个不太合格的审查员。

**萌萌**：好吧这点我被你说服了。但话说回来，那篇文章评论区有个观点我觉得挺好的——关键不是要不要用 AI，而是你得知道什么时候该关掉它自己想一想。

**芊悦**：对，就像健身一样，跑步机可以帮你调速度，但你得自己迈腿。用 AI 没问题，但偶尔得脱离舒适区，自己从零写点东西。

**萌萌**：哎说到这个，Dev.to 上也有篇相关的——"Surviving as a Human in the Age of AI"，AI 时代人类开发者的生存指南。看来全球程序员都在焦虑同一件事。

**芊悦**：而且还有那篇"八年想做，三个月用 AI 造出来"的故事。你看，同一个时代，有人焦虑技能退化，有人在用 AI 加速实现梦想。

**萌萌**：这就是硬币的两面嘛。我觉得关键是你是在"用" AI 还是在被 AI "用"。

**芊悦**：嗯，说得好。好了，接下来聊聊 Anthropic 这事，因为真的太戏剧化了。

**萌萌**：对对对！Claude Code 泄露源码的事还没完全消化呢，Anthropic 就开始搞商业化操作了——要求 Claude Code 订阅者用 OpenClaw 得额外付费。

**芊悦**：而且 Reddit 上讨论更激烈，说 Anthropic 直接不让 Claude Code 订阅用户用 OpenClaw 了。这事在社区里炸锅了。

**萌萌**：我觉得吧，Anthropic 有点着急了。你刚泄露了源码被全网讨论，转头就收紧商业策略，这时机也太巧了吧？

**芊悦**：我倒觉得这可能是早就计划好的，跟泄露事件未必有直接关系。但从用户角度看，体验确实不好——我交了钱用你的工具，你突然告诉我某些集成要加钱？

**萌萌**：就好比你买了个手机，用了半年突然告诉你，蓝牙功能要额外订阅才能用。离谱。

**芊悦**：不过从 Anthropic 的角度想，他们可能觉得第三方平台白嫖了他们的 API 算力。这种商业模式的博弈，以后在 AI 行业会越来越常见。

**萌萌**：嗯，这倒是。AI 工具的生态还在洗牌期，谁该收谁的钱，现在都没定论呢。

**芊悦**：对了，说到商业模式的迷惑操作，微软那个更搞笑——Copilot 的使用条款居然写"仅供娱乐用途"。

**萌萌**：哈哈哈哈哈我看到这条的时候笑死了！微软你可是把 Copilot 塞进 Office 全家桶的啊，现在告诉我这是"娱乐"？那我用 Copilot 写的工作报告算什么，脱口秀剧本吗？

**芊悦**：这其实是法律团队的防御性操作。AI 输出不可控，万一出了错，这个条款就是他们的免责盾牌。但确实很讽刺——一边卖给企业当生产力工具，一边说这只是玩具。

**萌萌**：典型的"既要又要"。不过你说，以后会不会所有 AI 产品的条款都变成这样？用出了问题，甩锅给"娱乐"俩字？

**芊悦**：很有可能。在法律法规跟上 AI 发展速度之前，这种擦边球条款只会越来越多。消费者得自己多留个心眼。

**芊悦**：好，快速过几条——Google Gemma 4 发布了，号称最智能的开源模型，而且换成了 Apache 2.0 许可证。

**萌萌**：Apache 2.0 这个很关键，意味着商业使用完全放开了。Google 在开源这块越来越大方了。

**芊悦**：GitHub 上 openscreen 爆火，两天涨了快三千星，开源屏幕录制工具，免费无水印。

**萌萌**：Screen Studio 的平替终于来了！我钱包表示感谢。

**芊悦**：Apple 批准了让 Nvidia eGPU 在 ARM Mac 上跑的驱动，Tiny Corp 开发的，主要面向 LLM 推理。

**萌萌**：等等，Apple 居然批准了 Nvidia 的东西？这是太阳从西边出来了吗？

**芊悦**：哈哈，说明 AI 推理的需求太大了，连 Apple 都不得不让步。还有 Suno AI 音乐版权的问题——AI 生成的山寨歌曲涌入流媒体平台，版权争议升级。

**萌萌**：这个问题今年肯定会越来越严重。音乐行业要是不赶紧想办法，以后各大平台全是 AI 歌曲，真人歌手的流量都要被稀释了。

**芊悦**：Perplexity 也出事了，被集体诉讼说把用户对话分享给 Meta 和 Google，号称的"隐身模式"形同虚设。

**萌萌**：我天...所以你以为你在隐身，其实你在裸奔？AI 时代的隐私问题真的让人头大。

**芊悦**：学术圈这边，北大提出了 DataFlex 框架，做数据驱动的 LLM 动态训练，还有一篇关于隐空间的综述论文都挺火的。

**萌萌**：日本那边也很有意思——机器人不是来抢工作的，是来填补没人想做的岗位。这个视角我觉得比欧美那种"AI 取代人类"的叙事健康多了。

**芊悦**：确实，不同文化对 AI 的态度差异真的很大。好了，总结一下今天的关键洞察——AI 工具在变强，但开发者不能变懒；AI 行业的商业模式还在混乱中寻找平衡；隐私问题会成为今年最大的雷区之一。

**萌萌**：每天 5 分钟，掌握科技圈大小事！我们明天见～
