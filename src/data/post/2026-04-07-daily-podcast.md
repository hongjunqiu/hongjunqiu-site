**芊悦**：嘿！大家好，欢迎收听《科技早知道》，我是芊悦～今天是4月7号，周二。

**萌萌**：我是萌萌！今天有两条消息我是真没忍住——Claude Code 被用户集体吐槽质量暴跌，还有 Adobe 偷改你电脑文件？这剧情我都编不出来，赶紧聊！

**芊悦**：哈哈好，咱们先从这个最炸的开始。Hacker News 上 Claude Code 那个 issue，707 分、446 条评论，社区直接炸锅了。

**萌萌**：等等，700 多分是什么概念？平时能上 300 的帖子就算热门了吧？

**芊悦**：对，这基本是"全民公愤"级别。大家反馈说二月更新之后，Claude Code 在复杂工程任务里变得不可用了。你想啊，之前能帮你重构整个模块的工具，突然连基本的代码生成都开始犯低级错误。

**萌萌**：这就像你常去的那家咖啡店，本来手冲做得贼好，突然换了个配方，拿铁变刷锅水了，你能不气吗？

**芊悦**：哈哈哈刷锅水，绝了。但我觉得这事背后有个更深的问题——AI 编程工具现在没有版本回退机制。用户被迫接受每次更新，好坏全靠运气。

**萌萌**：我不太同意诶。你说的回退听起来合理，但问题是这些大模型不是传统软件，你没法简单地"回到上个版本"，因为模型权重是一体的。

**芊悦**：嗯，技术上确实难，但产品层面可以做啊。比如保留旧模型 endpoint，让用户自己选。Anthropic 完全有能力做这个，只是选择不做。

**萌萌**：好吧，这点你说服我了。而且你注意到没，TechCrunch 还报了 Anthropic 要给 Claude Code 订阅者额外收费用 OpenClaw——一边涨价一边降质量，这操作也是没谁了。

**芊悦**：商业公司嘛，增长压力在那儿。不过说到信任问题，Hacker News 第二条热帖更猛——纽约客发了篇长文，标题直接问"Sam Altman 能被信任吗？"

**萌萌**：624 分！纽约客写 AI 圈的人物长文，这种组合本身就很有意思。以前他们写的都是文学家、政客，现在科技圈老板也够格被"解剖"了。

**芊悦**：这篇文章探讨的核心是权力集中的问题。一个人同时掌控着可能改变人类未来的技术和一家千亿估值的公司，这种权力浓度历史上很少见。

**萌萌**：怎么说呢，我反而觉得这种质疑有点马后炮。当年乔布斯不也是一个人说了算？扎克伯格到现在还是 Meta 的绝对控制者。硅谷一直就是这样啊。

**芊悦**：但 AI 不一样啊萌萌！手机做差了顶多不好用，AI 做差了——或者说做"太好了"——影响的是整个社会结构。这个量级的权力需要更多制衡。

**萌萌**：好吧好吧，你赢了，权力越大责任越大嘛。哎说到这个，Dev.to 上还有篇文章叫"Claude Code 源码泄露事件深度分析"，你看了没？

**芊悦**：看了！作者在分析这到底是事故、失误还是最强 PR。我倾向于认为是意外，但客观上确实给 Claude Code 带了一波流量。

**萌萌**：你这就太阴谋论了吧哈哈哈。对了，咱们聊点轻松的——Bram Cohen，就是发明 BitTorrent 的那位大佬，写了篇文章批评 Vibe Coding。

**芊悦**：444 分，说明很多人都有共鸣。他说 Vibe Coding 的狂热本质上是失控的 Dogfooding——就是 AI 公司的人自己用自己产品觉得很爽，然后推广给所有人，忽略了大部分人的场景完全不一样。

**萌萌**：这个比喻太精准了。就好比你是个大厨，觉得自己随便撒把盐就能出好菜，然后跟所有人说"做菜不需要量勺"——拜托，普通人真的需要量勺好吗！

**芊悦**：哈哈哈量勺比喻绝了。而且 Reddit 那边 r/ProgrammingLanguages 直接把 AI 生成的低质量项目给禁了，说明社区已经在反弹了。

**萌萌**：该禁！你去看看最近 GitHub 上多少项目 README 写得花里胡哨，配图精美得像产品发布会，代码一打开全是 AI 味儿的注释，跑都跑不起来。真的是金玉其外败絮其中。

**芊悦**：不过 GitHub Trending 上还是有好东西的。NousResearch 的 hermes-agent 今天涨了 1700 多星，"与你一起成长的 AI Agent"，开源的。

**萌萌**：还有 openscreen！免费开源的屏幕录制工具，Screen Studio 的替代品，无水印还能商用。今天涨了 1800 多星，比 hermes-agent 还猛。还有个 GitNexus，零服务器就能在浏览器里生成代码仓库的知识图谱，听起来很酷。

**芊悦**：开源真的在吃付费工具的午餐。接下来快速过几条——SpaceX 秘密提交了 IPO 申请，但隐瞒了财务数据，估值还在争议中。

**萌萌**：马斯克又搞神秘操作。Google Vids 2.0 上了 Product Hunt，视频协作工具加了 AI 编辑功能。

**芊悦**：Hugging Face 上有篇流式视频理解的论文挺有意思，还有一篇在探讨 Agent 能力对多模态智能的真正贡献。学术圈也在反思 Agent 到底有多大用。

**萌萌**：Dev.to 那篇"浏览器原生就能解决的 9 件事"我超推荐，很多前端开发者真的在过度工程化。

**芊悦**：对了，还有个好玩的——Adobe 被发现偷偷改用户的 hosts 文件来检测盗版！

**萌萌**：不是吧？！hosts 文件诶！那可是系统级的东西，你一个应用程序偷偷改这个，这不是流氓软件行为吗？

**萌萌**：这就像房东为了确认你住没住，不敲门问一声，直接把你家锁芯换了看你能不能开门——离谱！

**芊悦**：最后还有条暖心的——日本在用机器人填补没人想做的岗位，TechCrunch 说这证明物理 AI 已经准备好进入现实世界了。不是抢工作，是补缺口。

**萌萌**：这个叙事角度我喜欢，不是"机器人来了我们完蛋了"，而是"机器人来帮忙了"。日本老龄化严重，这种应用太合理了。
